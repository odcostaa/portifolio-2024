Portifólio

Olá, meu nome é Diogo, estou cursando Desenvolvimento de Sistemas na Etec de Hortolândia e tenho 17 anos. Este portfólio foi criado como parte de um projeto interdisciplinar, integrando as disciplinas de Programação Web (PW1), Técnicas de Programação Avançada (TPA) e Design Digital. Neste espaço, você encontrará os projetos que desenvolvi ao longo do curso, nos quais aplico os conhecimentos adquiridos em cada uma dessas áreas. Esses projetos refletem meu empenho em dominar conceitos fundamentais de programação, design e desenvolvimento de software, bem como a capacidade de integrar diferentes tecnologias para criar soluções eficientes e inovadoras.
